1.3. Contributing Tutorial

First download Guix using git 

git clone https://git.savannah.gnu.org/git/guix.git

Set the right environmental variables 

guix shell -D guix

Run the following 

./bootstrap
./configure --localstatedir=/var
make

We may add files however we want now. Then we will add the changes as a commit 

git add -A
git commit -am "Making commit."

'(+ 1 2)

(cons '- (cdr '(+ 1 2)))
