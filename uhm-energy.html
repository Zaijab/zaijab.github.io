2.2.1. Windows

Download the following software

  1. Putty
  2. Puttygen
  1. Generate Public and Private Key

    Use Puttygen to generate two files, a public key and a private key Send the public key to a server admin (Justin Delp, Zain Jabbar) The server admin will add the public key to home/$USERTOADD.ssh/authorizedkeys Use the following command to change Puttygen to OpenSSH: 

    ssh-keygen -i -f ~/dwn/public_test_key
  2. Connect via Putty

    Use the registry configuration to edit the Putty configuration Then edit private and public keys in settings

2.4. Tableau + Database

CREATE ROLE uhm_campus_energy_readaccess;
GRANT CONNECT ON DATABASE uhm2022 TO uhm_campus_energy_readaccess;
GRANT USAGE ON SCHEMA public to uhm_campus_energy_readaccess;
GRANT SELECT ON ALL TABLES IN SCHEMA public TO uhm_campus_energy_readaccess;
CREATE USER uhm_campus_energy_reader WITH PASSWORD '#anew=Sp!SL44pu95ube';
GRANT uhm_campus_energy_readaccess TO uhm_campus_energy_reader;

Option Name Option Value
Source Port 5432
Destination 128.171.46.101:5432
