1.1. GNU Guix

Guix is a functional package manager. That means that the operations done on the system are transactional. We must define three environments for the system to work properly.

Define the state of the operating system in the `operating-system` record. Define the state of the home environment in the `home-environment` record. Define the state of the installed files in the `manifest`.
