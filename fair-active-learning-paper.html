2. Introduction

A type of machine learning in which the learning algorithm queries a trusted source to label data points.

Machine learning is a very data hungry discipline. Models appreciate greater availability of data to learn from. This may be justified as follows:

  • Feed forward neural networks are proven to be universal function approximators. Using the Simple Function Approximation Theorem, we note that Decision trees and Boosting allows for Universal Approximation.
  • PAC learnable models use data to fit itself to the function in question under a reasonable time complexity and bounded error.

However, just because a model could learn better from more data does not entail what data would lead to the best approximation. PAC learning theory states that the bound is given by

Bound for arbitrary input data

There may be more reasonable data \( X_{1} \) that reduces the error more.

There are times in which labeling said data is time consuming or expensive. When given a dataset, we would like to know which data would result in a more effective model.

In a health care setting, if we have a patient with some subset of tests and an outcome to predict, then what is the current confidence in the prediction given now and what would be the most helpful tests to increase said confidence.

A review paper published in 2009 (Settles, Burr, 2009) gives an overview of the field. However, a number of papers have been published since then. We also want to place an emphasis on the role active learning has in the health care setting and on applications to reducing bias.

Active learning allows for a greater degree of automation in clinical tasks. Doctors are placed under a lot of time contraints [Citation Needed].

Fair Active Learning involves a dataset \( (X,y) \) which has labelled and unlabelled parts \( (X_{L}, y_{L}), (X_{U}, y_{U}) \) and we also have the sensitive attributes of the dataset \( (X_{s}, y_{s}) \)

The code is organized as follows, download a dataset in its entirety: df put the sensitive attributes in the multiindex of df, also return the labelled index
